# Quick cheatsheet

1. Keymapping
    * Keymap of choice:

        `dk-latin1`

    * Often used keys:

        | US Key | DK Key |
        |--------|------------|
        | `~`    | `ยง`        |
        | `|`    | `*`        |
        | `/`    | `-`        |
        | `-`    | `+`        |


    * Find all keymaps:

        `ls /usr/share/kbd/keymaps/**/*.map.gz`
    
    * Load keymap:

        `loadkeys <keymap>`
